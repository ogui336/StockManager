# StockManager

Um sistema de gestão de stock fictício desenvolvido em Java para consolidar conceitos práticos de Programação Orientada a Objetos (OOP).

## Etapas do Projeto
- [ ] Criação das classes de Modelo (Encapsulamento)
- [ ] Conexão com a base de dados SQLite (JDBC)
- [ ] Implementação do padrão DAO (Data Access Object)
- [ ] Construção da Interface Gráfica

## Tecnologias Utilizadas
* Java SE 21
* SQLite (em breve)
